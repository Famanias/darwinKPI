Overview

DarwinKPI is a performance management platform for organizations that need a centralized, data-driven way to track Key Performance Indicators. The system supports multiple isolated organizations, each with invite-code-based membership and role-based access control.

With DarwinKPI you can define KPIs, upload performance data, monitor trends through interactive charts, and export PDF reports — all in one place.

📈 Interactive Charts 📂 CSV & Excel Import 📄 PDF Reports 👥 Multi-org Support 🔒 Role-based Access 📋 Activity Audit Logs

Getting Started

There are three ways to get an account on DarwinKPI:

Option 1 — Create a New Organization

Use this if your organization is not yet on DarwinKPI. You will become the Admin of the new organization.

  1. 1Navigate to the Register page.
  2. 2Enter your name, email address, and a password (minimum 8 characters).
  3. 3Enter a unique Organization Name in the organization field.
  4. 4Submit — your account is created as the organization Admin.
  5. 5Find your Invite Code in Organization Settings and share it with colleagues.

Option 2 — Join an Existing Organization

Use this if your organization already has a DarwinKPI account and an Admin has given you an invite code.

  1. 1Navigate to the Register page.
  2. 2Enter your name, email, and password.
  3. 3Enter the Invite Code provided by your Admin (leave Organization Name blank).
  4. 4Submit — you join the organization with the User role by default.
Tip: Your Admin can later promote your role to Analyst or Admin from the User Management page.

Option 3 — Admin Creates You Directly

An organization Admin can create accounts on your behalf from the User Management panel. You will receive a temporary password of TempPass123! unless the Admin sets a custom one.

Logging In

Navigate to the Login page, enter your email and password, and you will be taken to your Dashboard.

User Roles

DarwinKPI uses three roles to control access within an organization. Each new member who joins via invite code starts as a User; Admins can reassign roles at any time.

Role Permissions
Admin Full access — manage users and roles, create and manage KPIs, import data, view all analytics, manage organization settings, view activity logs, and access the Admin Dashboard.
Analyst Create and manage KPIs, import performance data, view dashboard and analytics.
User View dashboard and analytics, import performance data.
Note: Only one role is assigned per user. Role changes take effect immediately and are recorded in the activity log.

KPI Management

KPIs are the core metric definitions in DarwinKPI. Each KPI belongs to your organization and is visible to all members.

Creating a KPI

Navigate to KPI Management in the sidebar. Click Add KPI and fill in the following fields:

  • Name — A short, descriptive name (e.g., “Monthly Revenue”).
  • Description — Optional context about what this KPI measures.
  • Unit — The unit of measurement (e.g., “USD”, “%”, “hours”).
  • Target Value — The goal or benchmark value for this KPI.
  • Frequency — How often data is reported: Daily, Weekly, or Monthly.
  • Visualization — Preferred chart type: Line or Bar.

Editing & Deleting KPIs

Each KPI in the list has Edit and Delete actions. Deleting a KPI also removes all associated performance data entries.

Warning: Deleting a KPI is permanent and cannot be undone. All historical data linked to that KPI will be lost.

Data Import

DarwinKPI accepts performance data uploaded as .csv or .xlsx files. Each upload is linked to a specific KPI.

File Format

Your file should have two columns:

  • date — The date of the data point (e.g., 2024-01-15 or January 15, 2024).
  • value — The numeric value for that date.
Tip: The first row should be a header row with column names. Extra columns are ignored.

Uploading Data

  1. 1Go to Data Import in the sidebar.
  2. 2Select the KPI you want to upload data for.
  3. 3Choose your .csv or .xlsx file.
  4. 4Click Import — the data is parsed and stored immediately.

Uploaded records are linked to your user account and visible to everyone in your organization once imported.

Dashboard

The Dashboard gives you a per-KPI view of performance over time. Each KPI you have data for appears as an interactive chart widget.

Time Range Filters

Use the time range selector above each chart to change the period displayed:

  • 7 Days
  • 30 Days
  • 90 Days
  • 1 Year
  • All Time
  • Custom Date Range — pick a start and end date manually.

Chart Types

Each chart renders as the visualization type set on the KPI (Line or Bar). The target value is displayed as a reference line so you can see performance relative to the goal at a glance.

Analytics

The Analytics page provides an aggregated, organization-wide view of all KPI performance data. It is designed for exploration and trend analysis across multiple metrics at once.

  • View performance trends for all KPIs side by side.
  • Filter by individual KPI or date range.
  • Identify patterns and compare metrics across different time windows.

Available to all roles: Admin, Analyst, and User.

Admin Dashboard

The Admin Dashboard is a high-level overview page exclusive to users with the Admin role. It summarizes organization-wide performance including:

  • Total KPIs defined and active data entries.
  • Recent data import activity.
  • Overall performance status across all KPIs.
  • Quick links to key administrative functions.

User Management

Available to Admin only. Manage all accounts within your organization from one place.

Viewing Users

The User Management page lists every member of your organization with their name, email, role, and status (Active or Inactive).

Adding a User

  1. 1Click Add User.
  2. 2Fill in the user’s name, email, and password.
  3. 3Assign a role: Admin, Analyst, or User.
  4. 4Submit — the account is created immediately.

Changing Roles & Status

Click Edit next to any user to update their role or toggle their account status between Active and Inactive. Inactive users cannot log in.

Organization Settings

Available to Admin only. View and manage your organization’s profile and invite code.

Invite Code

Your organization has a unique invite code. Share this code with anyone you want to add — they enter it on the Register page to join your organization automatically.

If you believe the code has been compromised, click Regenerate Invite Code to issue a new one. The old code becomes invalid immediately.

Warning: Regenerating the invite code invalidates the previous one. Anyone who has not yet registered using the old code will need the new one.

Reports & Downloads

DarwinKPI lets you export KPI performance data as a formatted PDF report. Reports are generated on demand and downloaded directly to your device.

What a Report Includes

  • Branded header with organization name and report date.
  • KPI details: name, description, unit, target, and frequency.
  • Formatted data table with all performance entries in the selected time range.
  • Page numbers.

Generating a Report

  1. 1Open the Dashboard and navigate to the KPI you want to export.
  2. 2Set the desired time range using the filter controls.
  3. 3Click the Download PDF button on the KPI widget.
  4. 4The report will download automatically to your browser’s default download folder.

Activity Logs

Available to Admin only. The Activity Logs page provides a full audit trail of user actions within your organization.

What Is Logged

  • User logins and logouts.
  • KPI creation, edits, and deletions.
  • Data import events (which user imported data for which KPI).
  • User account changes (role updates, status changes, new accounts).
  • Organization settings changes (e.g., invite code regeneration).

Filtering Logs

Use the user filter dropdown to view activity from a specific team member. This is useful for accountability reviews or investigating unexpected changes.
